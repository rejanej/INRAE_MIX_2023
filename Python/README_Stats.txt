UTILISATION DE LA FENETRE GRAPHIQUE
-La première fenêtre graphique permet de sélectionner les données que l'on souhaite étudier par la suite.
On choisit tout d'abord le fichier excel qui nous intéresse, puis les feuilles à considérer, et pour finir les variables que l'on souhaite conserver dans ces feuilles. La variable "DATE" doit systématiquement être choisie et une seule fois.
Une fois le bouton "Traiter" pressé, une seconde fenêtre graphique apparait, permettant de réaliser différents type de traitements sur les images.

-5 onglets sont disponibles.
    Le premier "Visualiser Données" permet d'afficher toutes les données pour chaque variable sélectionnée en fonction de la date.


    Le second onglet "Outiliers" met en avant les valeurs aberrantes dans le jeu données, et offre la possibilité de visualiser les données sans les outliers.

    Le troisième onglet "Visualiser Courbes des Variables" offre la possibilité de visualiser sous forme de courbe chaque variable, ou plusieurs, sur un unique graphique. L'utilisateur peut choisir les variables qu'il souhaite visualiser, mais aussi la plage de date souhaitée. Le bouton "Afficher Courbes Variables" permet de visualiser les données d'une ou plusieurs variables sur l'ensemble des données, tandis que le bouton "Afficher Courbes Variables avec Data Range" permet de visualiser les données d'une ou plusieurs variables sur la plage de date choisie. Les dates doivent être rentrées au format yyyy:mm:jj, le jour et le mois pouvant être omis.

    Le quatrième onglet "Visualiser Corrélations" permet d'analyser d'éventuelles corrélations entre les variables. Le bouton en haut doit être coché pour avoir ensuite la possibilité de choisir comme dans l'onglet précédent la plage de date d'étude souhaitée en décochant le bouton (valable pour tous les traitements de la fenêtre).
    Il est ensuite possible de croiser deux variables au choix pour afficher le nuage de point correspondant (bouton "Afficher Données")
    En sélectionnant les variables d'études désirée sur la droite, l'utilisateur peut ensuite afficher une matrice des corrélations, un diagramme de dispersion (fonctionnel uniquement avec ChloroA dans les variables choisies) , ainsi qu'une cercle de corrélation issue d'une ACP sur les données.
    Le bouton "Afficher OLS Reg" permet de visualiser d'éventuelles corrélations entre les variables avec la quantité de ChloroA ( non fonctionnel )
    On peut ensuite visualiser tout en bas de la fenêtre des cross-corrélations entre les séries temporelles de deux variables au choix.
    L'onglet "Réduction Dimensionnalité" 
    possède le même bouton à cocher que la fenêtre précédente. Il permet de visualier le UMAP sous différents aspects en fonction des variables choisies. Les paramètres nb neighbors et min dist doivent être renseignées.
